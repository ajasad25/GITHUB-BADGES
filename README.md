# 🏆 GitHub Profile Achievements — Complete Guide (2025)

A complete, up-to-date guide to every GitHub profile achievement badge — how to earn them, tier requirements, and which ones are still available.

## All Achievements

| Badge | Name | How to Earn | Tiers |
|---|---|---|---|
| 🌟 | **Starstruck** | Create a repo with 16+ stars | 16 / 128 / 512 / 4096 |
| 🦈 | **Pull Shark** | Merge 2+ pull requests | 2 / 16 / 128 / 1024 |
| 🧠 | **Galaxy Brain** | 2+ accepted answers in Discussions | 2 / 8 / 16 / 32 |
| 🫛 | **Pair Extraordinaire** | Co-author a merged pull request | 1 / 10 / 24 / 48 |
| 🤠 | **Quickdraw** | Close an issue/PR within 5 min of opening | No tiers |
| 🟡 | **YOLO** | Merge a PR without code review | No tiers |
| 💖 | **Public Sponsor** | Sponsor an open source developer | No tiers |

## How to Earn Each Badge

### Starstruck
Create a public repository that other people find useful. When it reaches **16 stars**, you unlock the badge. Higher tiers at 128, 512, and 4096 stars.

### Pull Shark
Open pull requests and get them merged. You need **2 merged PRs** for the base badge. Works on any repo (including your own).

### Galaxy Brain
Answer questions in **GitHub Discussions** (Q&A category). When the question author marks your answer as accepted, it counts. You need **2 accepted answers**.

### Pair Extraordinaire
Add a `Co-authored-by` trailer to a commit message in a merged PR:
```
Co-authored-by: name <email@example.com>
```

### Quickdraw
Open an issue or pull request and close it **within 5 minutes**.

### YOLO
Merge a pull request **without any code review** (no approvals, no review comments).

### Public Sponsor
Go to [GitHub Sponsors](https://github.com/sponsors) and sponsor any developer or organization. Even a $1/month sponsorship counts.

- **Heart On Your Sleeve** — React with ❤️ on GitHub
- **Open Sourcerer** — Have PRs merged in multiple public repos you don't own
