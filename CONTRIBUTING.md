# Contributing

Thanks for wanting to help improve this guide!

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your changes: `git checkout -b fix/update-badge-info`
3. **Make your changes** and commit them with clear messages
4. **Open a Pull Request** describing what you changed and why

## What We Need Help With

- **Badge requirement updates** — If GitHub changes how a badge is earned or its tier thresholds, please open an issue or PR.
- **New badges** — If GitHub releases a new achievement, add it to the appropriate section.
- **Image updates** — If badge artwork changes, update the files in the `Media/` directory.
- **Translations** — Help make this guide accessible in more languages.

## Guidelines

- Keep descriptions concise and accurate
- Include sources/links when reporting changes to badge requirements
- Test that all image paths render correctly before submitting
- Follow the existing formatting patterns in the README

## Reporting Issues

If you notice incorrect information, broken images, or missing badges, please [open an issue](../../issues/new) with details.
